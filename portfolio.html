My Project

Student Management System

Project Description

Student Management System is a web-based application developed to manage student records efficiently. The system allows users to register new students, view student details, update information, delete records, and search students by name, department, or city. The application helps reduce manual paperwork and provides a simple interface for managing student data.

Features

  • Student Registration
  • View All Students
  • Update Student Details
  • Delete Student Records
  • Search by Name
  • Search by Department
  • Search by City
